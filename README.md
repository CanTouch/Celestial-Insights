# Celestial-Insights
A personalized astrology and horoscope app powered by Gemini AI. Features daily readings, moon phase tracking, compatibility calculator, and a celestial events calendar.
Feature List: Detailed breakdown of daily horoscopes, moon tracking, and compatibility tools.
Tech Stack: Highlights React 19, Vite, Tailwind CSS 4, Firebase, and Gemini AI.
Setup Instructions: Step-by-step guide for cloning, installing dependencies, and configuring environment variables.
Prerequisites: Links to get Gemini API keys and set up Firebase.
